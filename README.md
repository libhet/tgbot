# Exercise Tracker

Инструмент для ведения прогресса по упражнениям с учётом повторов. Поддерживает статусы (`pending`, `in_progress`, `completed`), визуализацию прогресса и отчётность.

## Возможности

- Создание упражнений с целевым количеством повторов.
- Автоматический статус в зависимости от количества выполненных повторов.
- Генерация «кнопок» вида «Повтор #n» для интеграции в UI/бота.
- Табличный список и календарное представление прогресса.
- Хранение истории и перевод упражнения в «выполнено» при достижении цели.
- Отчёты по текущей неделе и месяцу.

## Установка зависимостей

```bash
pip install -e .[dev]
```

## Использование CLI

```bash
python -m exercise_tracker.cli add "Отжимания" 3
python -m exercise_tracker.cli mark "Отжимания"
python -m exercise_tracker.cli list
python -m exercise_tracker.cli calendar --days 14
python -m exercise_tracker.cli report
```

## Тесты

pytest
